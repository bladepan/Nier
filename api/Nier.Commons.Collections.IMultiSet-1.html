    Interface IMultiSet<TValue>

    A collection that supports order-independent equality, constant time look up. It is like , but may have duplicate elements. Similar to MultiSet in Guava or Bag in Apache Commons.

    It is useful to keep occurrences of items, i.e. counters. It provides constant time of increase, decrease, lookup and update counts of items.

    It uses object.Equals(obj1, obj2) to determine if two objects are the same, unless specified otherwise by implementation.
